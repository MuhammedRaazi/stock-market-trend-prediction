---

## Project Overview
This project focuses on directional classification, not price prediction.  
It predicts whether a selected stock is likely to show:
- Upward movement
- Downward movement
- Strong directional movement based on volatility and trend strength

The application is built with Python, Machine Learning, and Streamlit and provides an interactive dashboard for predictions.

---

## Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- XGBoost
- yFinance
- Matplotlib, Seaborn
- Streamlit

---

## Features
- Historical data fetching using `yfinance`
- Feature engineering with technical indicators
- ML model trained using XGBoost
- Confidence score for predictions
- Interactive Streamlit dashboard
